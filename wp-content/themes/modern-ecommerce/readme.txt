== Description ==

Modern Ecommerce theme is a suitable theme for business, shop, store, book shop, fashion store, shoes, watches, grocery shop, apparel store, eyewear, boutique and electronics and for any online shop. You can create your online store easily with all the basic options. The theme is modern, elegant, sophisticated, minimal and retina-ready in design. It is based on bootstrap framework which makes it very handy to use. It is translation-ready and supports RTL layout. It is optimized for speed hence loads without any glitches. You can add short codes to it if you know coding. It is written with clean and secure codes. It is user-friendly theme that offers many options for customization and personalization. It is SEO-friendly too which will help your website rank soon on major search engines like Google. The responsive layout of the theme makes it mobile-friendly therefore it fits nicely in any device.
